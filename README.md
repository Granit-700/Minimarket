# Minimarket

## Описание

Пример простого интернет минимаркета с разделением на каталоги. \
Присутствует полный CRUD-функционал

## Стек

- HTML
- CSS
- TypeScript
- React-Router-Dom
- Axios
- Zustand
- Firebase RealTime Database
- React-Toastify
- Vite

## Запуск

### Скачивание репозитория

```sh
git clone https://github.com/Granit-700/Minimarket.git
```

### Создание .env файла

```
VITE_FIREBASE_URL=https://your_firebase_realtime_databese_url
```

### Установка зависимостей

```sh
npm i
```

### Старт в DEV-режиме

```sh
npm run dev
```
